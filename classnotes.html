The first lesson was a basic intro to the web, and then got into HTML stuff. We learned about links to stuff: links to awesome educational sites and classic sites everyone knows. Links to sweet cat videos and that sort of thing. This paragraph conveniently also shows attribute tags and both block and inline elements. And of course a photo.

Second lesson

The second lesson introduced stylesheets which are pretty mindblowing and give you huge power to customize how your site looks. While everything in HTML is based on blocks, you have so much power with style sheets you can really make your page look like whatever you want and the blocks help organize and think about what you're doing. I was stuck on a plane during this section so I ended up picking apart a cached site that was really pretty and stealing elements of their layout. A few other things I learned:

  • CSS is cascading, with the most specific element having the most power.
  • You seem to be able to combine styles for example span class="greenttext boldtext" would make green bold text: 
    .greentext
{
color: green;
}
.boldtext
{
font-weight: bold;
}
