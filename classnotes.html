Fourth paragraph

After playing with stylesheets a little, I discovered several cool things:

  • There seem to be two ways to do styles, the first is specific and only applies when you exactly phrase the class name the same, for example the below will only apply to divs with "class=cooltitle": 
    div.cooltitle {
	color: #C60;
	background-color:#FFFFDD;
}
  • This second way seems to apply to anything with the root phrase of the class, for example the below will only apply to divs with "class=cooltitle1", "cooltitle2" or "cooltitleb": 
    div, .cooltitle {
	color: #C60;
	ackground-color:#FFFFDD;
}
  • This is pretty neat. I wouldn't have dug into css quite so deeply yet but I'm stuck on a plane with no internet connection so I can't continue the lesson. Luckily I had samples to use for inspiration.
